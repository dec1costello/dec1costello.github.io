MLB Prospect Pipeline

Overview

This project involved creating ETL pipelines that centralize MLB and MiLB data for scouting and player development decision-making.

Tools & Technologies

  • Python (Airflow, Pandas, SQLAlchemy)
  • PostgreSQL + Cloud Storage
  • Dashboards built with Streamlit

System Overview Diagram

Impact

The pipeline enabled scouts and analysts to view updated stats and advanced metrics from multiple vendors in one interface. It reduced data lag from days to hours.
